#### Váriaveis
- Podem começar com letra, $ ou _
- Não podem começar com números
- É possivel usar letrar ou números
- É possivel usar acentos e simbolos
- Não podem conter espaços
- Não podem usar palavras reservadas

#### Dicas

- Minusculas
